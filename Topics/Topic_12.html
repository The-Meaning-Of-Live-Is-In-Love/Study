Главная

sort (anArray, n)

// Сортировка массива в возрастающем порядке.

Предусловие: переменная — «anArray» — является массивом, состоящим из n целых чисел; 1 <= n <= MAX_ARRAY, где MAX_ARRAY — глобальная константа, задающая максимальный размер массива — «anArray».

Постусловие: anArray[0] <= anArray[1] <= ... <= anArray[n - 1]; число — «n» — не меняется.
